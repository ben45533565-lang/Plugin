# OptiFlow Plugin
Bu eklenti, sunucu performansını optimize etmek ve oyuncu verilerini (envanter, sağlık, açlık, ekonomi) güvenli bir şekilde saklamak için geliştirilmiştir.

## Özellikler
- **Veri Kaydı:** Oyuncu çıktığında envanter, sağlık, açlık ve ekonomi verilerini otomatik kaydeder.
- **Vault Desteği:** Ekonomi verileri için Vault entegrasyonu mevcuttur.
- **Hafif Çalışma:** Sunucu kaynaklarını yormadan yüksek verimlilikle çalışır.

## Kurulum
1. `Plugin-1.0.jar` dosyasını indirin.
2. Sunucunuzun `plugins/` klasörüne atın.
3. Sunucuyu veya dünyanızı yeniden başlatın veya `/reload` komutunu kullanın.

## Gereksinimler
- **Vault:** Ekonomi özellikleri için gereklidir.
- **Spigot/Paper:** 1.20+ sürümleri desteklenmektedir.
